Gradle构建入门

发表于 2017-12-10 | 分类于 spring |
字数统计: 491 字 | 阅读时长 ≈ 2 分钟

在开发Spring Boot应用的时候，除了Maven以外，Gradle也是一个不错的构建选择。 IDEA 默认使用Gradle工具做Android的构建程序，在开发APP的时候就必须要使用到Gradle了。 这里简单记录一下使用的方法，以及跟IDEA的集成方式。

IDEA中使用

IDEA默认就支持Gradle，如果创建新的工程，只需要在向导中选择Gradle工程即可，后面的跟maven一样。 如果是导入已经存在的Gradle工程，就选择 New -> Project from existing sources，然后选中build.gradle即可。
