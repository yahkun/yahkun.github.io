启动参数说明：

  1. -l 指定绑定监听到哪个IP
  2. -u 用户名参数，当未设置-u参数，且计算机用户名为^[a-zA-Z0-9]+$时，使用计算机用户名作为idea用户名
  3. -p 参数，用于指定监听的端口
  4. -prolongationPeriod 指定过期时间参数
